# Specs Auxiliares

Os arquivos desta pasta existem como **referência estrutural auxiliar**.

Uso permitido:

- detalhar layout e componentes
- inspirar a forma de escrever specs futuras
- ajudar no desenho de módulos evolutivos

Uso não permitido:

- substituir `docs/01-15` como regra de negócio
- redefinir segurança, auth, grants, roles ou backend crítico
- introduzir domínio funcional conflitante com Mariana Explica

Em caso de conflito, a fonte oficial continua sendo:

- `docs/01-visao-geral.md` até `docs/15-plano-de-implementacao.md`
